# Портфолио моих проектов в области Data Analysis

| Проект | Описание | Инструменты |
|----------------|---------|----------------|
| [Исследование компьютерных игр](https://github.com/OlegDze/public/blob/main/Computer_games_analysis.ipynb) | Интернет-магазин «World of games» продаёт по всему миру компьютерные игры. Из открытых источников доступны исторические данные о продажах игр, оценки пользователей и экспертов, жанры и платформы (например, Xbox или PlayStation). | `Python, pandas, matplotlib, stats`, t-критерий Уэлча, корреляционный анализ|
